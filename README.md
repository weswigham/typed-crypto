typed-crypto [![Build Status](https://travis-ci.org/weswigham/typed-crypto.svg)](https://travis-ci.org/weswigham/typed-crypto)
============

TypeScript typings for the core crypto library

Usage
=====

First, add it to your `package.json` and install it:
```cmd
npm install --save typed-crypto
```

Then use it just as you would the crypto library:
```ts
import * as crypto from "typed-crypto";
```
